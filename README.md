# SSD1305

SPI_CS -->  IOID_20
SPI_DC  --> IOID_9
IC_RST --> IOID_13
SPI_SDA --> IOID_14
SPI_SCK --> IOID_10

I did the project on Code Composer Studio and used CC1350 Launchpad.
